项目白皮书：Proomet - MacOS 提示词管理App
1. 项目名称
Proomet
2. 核心目标
本地优先的MacOS提示词管理工具，帮助用户高效创建、组织、搜索、版本管理和复用AI提示词，提升提示词工程效率。
3. 提示词管理方式

创建与编辑：新建提示词，支持富文本编辑、变量占位符（{{variable}}）。在编辑页面选择分类（单选）和标签（多选）。
模块化构建器：拖拽拼装模块（角色、任务、上下文、格式、示例、约束），快速生成结构化提示词。
存储：默认本地向量数据库，支持全文+语义存储；可选iCloud同步。
组织：分类（默认“常用”“人设” + 用户自定义增删改）、标签（用户手动在编辑页添加多个）、最近使用列表。
版本控制：提示词演化树，支持分支（fork）、合并（merge）、回滚历史版本。
搜索：语义搜索 + 关键词 + 按分类/标签/最近使用过滤。
复用：一键复制、Shortcuts调用、菜单栏/Spotlight快速插入。
统计：记录使用次数、最后使用时间，AI基于统计给出优化建议（例如“添加few-shot可提升一致性”）。

4. 核心功能特色

语义搜索 + AI自动标签
提示词演化树（版本分支、fork/merge/回滚）
模块化拖拽构建器（角色/任务/格式/示例等积木拼装）
使用统计 + AI优化建议
MacOS原生集成（菜单栏快捷键、Spotlight搜索、Shortcuts调用）

5. 侧边栏结构

区域1：所有提示词（固定置顶）
区域2：分类（常用、人设 + 用户自定义，支持增删改） + 最近使用（置于上方）
区域3：标签（用户手动在提示词编辑页添加多个，动态显示）

6. 其他

本地向量数据库存储
可选iCloud同步
全键盘操作
